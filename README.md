
# Enso: Effortless Simultaneous Command Execution for Development Servers

In modern 

## features

* Simple
* Few dependencies

## Install

```bash
npm install --save-dev enso
```

```bash
yarn add -D enso
```

```bash
pnpm install --save-dev enso
```

## Usage

```bash
enso -- echo "Hello" -- echo "Good Bye"
```

`--`で実行したいコマンドをつないで実行するだけです。
